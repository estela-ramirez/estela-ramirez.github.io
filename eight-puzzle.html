8 puzzle

Solves 8 puzzle with 3 different search algorithms

This program solves the 8 puzzle using

  1. Uniform Cost Search
  2. A* with the Misplaced Tile heuristic
  3. A* with the Euclidean Distance heuristic

It shows the steps the algorithm takes to find a solution. Once the solution is found, the solution path is printed. You can use the default puzzle or input your own
